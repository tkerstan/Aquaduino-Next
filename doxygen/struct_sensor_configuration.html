char SensorConfiguration::data[bufferSize]

This buffer may hold the user data of the component. Only the component itself can interpret it using its Serializable::deserialize method. It is filled using the Serializable::serialize method of the component.

The size is specified by the macro SERIALIZATION_BUFFER
