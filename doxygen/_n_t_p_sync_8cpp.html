time_t NTPSync ( )

This routine contacts the NTP server specified in sntpServerIP. It waits 1 second for a response. If there is no response a timeout occurs and the routine exits and returns 0 as time.

Returns
Time since 1970 in seconds + timeZoneOffset in seconds
